### 2. 工作信息

#### 2.1 任职记录

| 字段 | 录入方式/数据源/说明 |
| --- | --- |
| 生效日期 | 选择，年月日 |
| 创建日期 | 系统生成/冗余记录字段 |
| 入职日期 | 选择，年月日 |
| 司龄 | 不可编辑，根据集团工龄开始日期计算得出 |
| 是否重新雇佣 | 选择，是/否 |
| 集团责任制开始日期 | 选择，年月日 |
| 集团工龄开始日期 | 选择，年月日 |
| 供应商 | 选择，数据源：数据字典「供应商」 |
| 试用期期限 | 选择，数据源：数据字典「试用期期限」（1个月/2个月/3个月/4个月/5个月/6个月） |
| 预计转正日期 | 不可编辑，冗余记录字段：入职日期 + 试用期期限 |
| 实际转正日期 | 选择，年月日 |
| 操作 | 选择，三字段联动；参考「员工职务数据异动类型数据源」中的「操作」 |
| 原因 | 选择，三字段联动；参考「员工职务数据异动类型数据源」中的「操作原因」 |
| 原因子项 | 选择，三字段联动；参考「员工职务数据异动类型数据源」中的「原因子项」 |
| 职务指示 | 选择：主要职务/次要职务 |
| 法人实体 | 选择，数据源：数据字典「法人公司」 |
| 部门 | 选择，数据源：组织数据 |
| 岗位 | 选择，数据源：岗位数据 |
| 岗位序列 | 不可编辑，选择岗位后带出岗位序列 |
| 职级 | 选择岗位后带出岗位序列/岗位职级；若岗位数据没有岗位职级，可手动选择修改；数据源：数据字典「岗位职级」 |
| 合同地点 | 选择，数据源：数据字典「合同地点」 |
| 工作地点 | 选择，数据源：数据字典「工作地点」 |
| 责任制 | 选择，是/否 |
| 审批权限 | 选择，数据源：数据字典「审批权限」 |
| 员工组 | 选择，参考本文档「员工组/员工子组数据源」 |
| 员工子组 | 选择，与员工组联动；参考本文档「员工组/员工子组数据源」 |
| 该岗位开始日期 | 不可编辑，冗余记录字段：职务数据上岗位编码发生变更的职务数据生效日期 |
| 在岗时间 | 不可编辑，冗余记录字段：当天 - 该岗位开始日期 + 1 |
| 员工性质 | 选择，数据源：数据字典「员工性质」（生产类/科研类/管理类/营销类/财务类/采购类） |
| 集团属性分级 | 选择，数据源：数据字典「集团属性分级」（一线操作人员/专业技术人员/中层管理者/中高层管理者/基层管理者/高层管理者） |
| 发薪公司 | 选择，数据源：数据字典「发薪公司」 |
| 成本归属法人 | 选择，数据源：数据字典「法人公司」 |
| 真实离职原因(HRBP) | 文本，手填 |
| 真实离职原因子类(HRBP) | 文本，手填 |
| 交接人 | 选择，数据源：员工 |
| 离职去向 | 文本，手填 |
| 是否启动竞业限制-公司建议 | 选择，是/否 |
| 是否给薪 | 选择，是/否 |
| 薪资组 | 选择，数据源：数据字典「薪资组」 |
